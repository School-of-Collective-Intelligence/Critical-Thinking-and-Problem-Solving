# Critical Thinking and Problem Solving

This module on **Critical Thinking** is offered by the **School of Collective Intelligence**.

## Course Duration

**20 hours total**  
(10 sessions × 2 hours each)

## Course Description

This course sharpens reasoning skills and exposes where reasoning breaks down.

We begin with the foundations of argumentation and the logic underlying strong claims. We then examine how cognitive biases distort judgment, how statistical errors and causal confusion can mislead even careful thinkers, and why different kinds of “why?” questions require different kinds of answers.

Students learn to diagnose logical fallacies in academic and public discourse and apply these tools to evolutionary puzzles (e.g., altruism, costly rituals, harmful cultural practices), where intuition often fails and structured reasoning is essential.

The course concludes with synthesis and problem solving: integrating ideas across domains, reasoning collectively through platforms such as **AIdeator**, and using AI tools to organize and pressure-test thinking.

## Course Format

Each 2-hour session includes:

- A short framing lecture (45 minutes)  
- In-class activities: *Critical Thinking Puzzles*  
- Class discussion
